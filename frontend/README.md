# Solana Counter dApp Frontend

This is a Next.js frontend for a simple counter dApp on the Solana blockchain. It allows users to interact with the Anchor counter program.

## Features

-   Connect to a Solana wallet.
-   Initialize a new counter account.
-   Increment the counter's value.
-   Reset the counter's value to zero.

## Getting Started

### Installation

1.  **Clone the repository**
2.  **Install Dependencies**
    ```shell
    npm install
    ```

### Running the Development Server

To start the development server, run:

```shell
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

-   **Framework**: Next.js
-   **Styling**: Tailwind CSS & Shadcn UI
-   **Solana Integration**: Gill Solana SDK & Wallet UI
